First Step

It was implemented a button that runs a Javascript code (compressed by Google Closure Compiler) to show a map bellow. This map should provide your device's location which you are accessing this website. If it's not compatible it will show the location of University of Toronto St. George campus

Second Step

On the screen bellow it was implemented HammerJS, so it can recognize horizontal swapping and pinch gesture. If you swap the screen horizontally, it will change the color, and if you pinch the screen, it will change the size of the screen

Third Step

This page was built to be completely different for different screen dimentions, try changing your mobile device orientation or resizing your browser screen size on your computer to try it.
